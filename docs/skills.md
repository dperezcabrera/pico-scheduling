# AI Coding Skills

[Claude Code](https://code.claude.com) and [OpenAI Codex](https://openai.com/index/introducing-codex/) skills for AI-assisted development across the pico ecosystem:

```bash
curl -sL https://raw.githubusercontent.com/dperezcabrera/pico-skills/main/install.sh | bash
```

The `pico-conventions` background skill teaches the assistant this module's
API surface and invariants; `AGENTS.md` and `CLAUDE.md` in the repo carry the
project-level rules an assistant must respect.
